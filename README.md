# MLOps Lab CIE

This repository contains the implementation of an end-to-end MLOps pipeline for UrbanRide fare prediction.

## Structure
- MLOPs_Lab_CIE/
  - data/ → Training and new datasets
  - src/ → Training, tuning, registration, retraining scripts
  - results/ → JSON outputs for all steps

## Tasks Completed
1. Experiment Tracking & Model Comparison
2. Hyperparameter Tuning
3. Model Registration
4. Retraining Pipeline

## Tools Used
- Python
- Scikit-learn
- MLflow

## Output
All results are stored as machine-readable JSON files inside `results/`.
